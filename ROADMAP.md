# Roadmap

This roadmap tracks public work around the hermes-agent.us website and product surface. It does not represent the upstream Nous Research runtime roadmap.

## Now

- Keep the deployment guide accurate for local, Docker, VPS, cloud, and managed hosting paths.
- Improve the managed hosting page with clearer launch, entitlement, and runtime status messaging.
- Keep Open WebUI integration guidance practical for users who want a browser-based frontend.
- Collect recurring setup questions through GitHub Issues and Discussions.

## Next

- Expand troubleshooting notes for common self-hosting failures.
- Add clearer upgrade guidance from short managed hosting access to longer-lived workspaces.
- Publish more comparison content for users evaluating persistent agent runtimes.
- Improve public release notes for site and managed-hosting changes.

## Later

- Add more deployment examples for team and server workflows.
- Add public templates for issue reports and environment checklists.
- Evaluate more managed-hosting controls once the core hosted workspace flow is stable.

## Not In Scope

- Forking or mirroring the upstream Hermes Agent runtime source.
- Publishing private managed-hosting infrastructure code in this repository.
- Handling private account, billing, or credential support through public issues.
