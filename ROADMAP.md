# Hermes Agent Public Roadmap

This roadmap summarizes product directions we are evaluating or actively improving for [hermes-agent.us](https://hermes-agent.us). It is not a promise of delivery dates, pricing, model access, or final feature scope.

## Now

- Keep deployment guidance accurate for local, Docker, VPS, cloud, and managed hosting paths.
- Improve the managed hosting page with clearer launch, entitlement, and runtime-status messaging.
- Clarify Open WebUI and OpenAI-compatible API integration context.
- Keep comparison pages practical and caveated rather than overstated.

## Next

- Add clearer troubleshooting flows for common deployment failures.
- Improve hosted WebUI expectations for first-run users.
- Expand comparison notes where runtime, memory, skills, and API compatibility differ.
- Document where the public website ends and upstream open-source runtime docs begin.

## Later

- Evaluate public status notes for managed hosting incidents or maintenance.
- Add deeper operator checklists for VPS and cloud deployments.
- Improve examples for Open WebUI client configuration.
- Document caveats around third-party runtime changes and upstream project scope.

## Product Principles

- Do not represent this website as the upstream Nous Research runtime roadmap.
- Deployment claims should stay tied to tested public documentation.
- Managed hosting should be described clearly without hiding operational boundaries.
- This repository is for public product feedback, not private production source code.
